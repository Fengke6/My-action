# 自动编译&发布

收录多个常用项目并进行编译，编译后自动回传到下载站，方便用户直接下载想要的成品。

如果下载站挂了 你也可以直接在[Github Actions](https://github.com/1orz/My-action/actions)里直接下载。(需要登录Github账号才能下载)

列表的所有编译成品均自动上传到下载站。

国内下载站(推荐) https://down.cloudorz.com/

备用下载站(日本) https://down.ssrc.win/
